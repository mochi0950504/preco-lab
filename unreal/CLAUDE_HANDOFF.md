## 0. 施工總順序建議
1. 確認 C++ 模組已編譯（指南 §1），DefaultEngine.ini 的 Lumen/PPV 生效
2. 建「標準層」Level Instance（§2 走廊＋§3 樓梯間）→ 複製為 2F–10F
3. 大廳（§4）→ 外觀（§5）→ 燈光與後處理（§7）
4. 放置互動 Actor（§6）→ 鬼（§8）→ 開場 Sequence（§9）
5. 每完成一段用視口截圖工具自我驗收，再進下一段

## 2. 標準層走廊（2F–10F 相同）
- 走廊：x ∈ [-13, +13]，寬 2.2（z ∈ [-1.1, +1.1]），紅地毯走道＋維多利亞牆紙（上 2/3）＋深木護牆板（下 1/3）＋簷線/踢腳線
- 客房槽位：兩側各 8 間，門中心 x = -11.2, -8, -4.8, -1.6, 1.6, 4.8, 8, 11.2（間距 3.2）
  - **南側 x=±1.6 兩槽不設門**（電梯佔位）
  - 房號跳過 13：南側 n01–n08、北側 n09–n12, n14–n17（n=樓層）
- 電梯：南牆 x ∈ [-1.3, 1.3] 凹間，雙滑門，內廂 1.7×1.5
- 壁燈：x = -9.6 起每 3.2m 交錯兩側，高 2.25m，約 40% 故障不亮，其餘隨機閃爍（§7）
- 10F 特例：所有門釘木板；北側 x=0 為「封牆」——牆上有門框痕跡，集滿證據後浮現 1013 門（用 AHVDoorActor 的 1013 模式）

## 3. 樓梯間（每層兩端鏡像，x=±[13, 16.8]，z ∈ [-4.2, 1.2]）
- 低平台 z ∈ [-1.0, 1.2]（與樓層同高）；走廊經 x=±13 的門洞（z ∈ [-0.9, 1.1]）進入
- A 段：外側車道 x ∈ [±15.3, ±16.7]，z -1.0 → -3.2 上升 1.7m（10 階）
- 中平台 z ∈ [-4.2, -3.2]（+1.7m）
- B 段：內側車道 x ∈ [±13.1, ±14.8]，z -3.2 → -1.0 再升 1.7m 抵上層
- 車道間隔牆 x=±15.0（厚 0.3）
- 外牆窗（x=±16.8，月光）每層一扇
- 10F：z<-1 區以圍欄封住（梯井到頂）；圍欄牆上是釘死的天台門＋門前**一雙擺正的舊布鞋**（劇情關鍵道具，可互動查看）

## 4. 大廳（1F）
- 開放空間 x ∈ [-13,13]、z ∈ [-6,6]，大理石棋盤地坪
- 櫃台：x ∈ [-4,4]，z=-3.4，高 1.1，後方鑰匙牆
- 大理石圓柱 ×4：(±7, ±3)，半徑 0.45
- 電梯島：中央 z ∈ [1.05, 2.75]，x ∈ [-1.7,1.7]，雙門朝北（中空可見內廂）
- 立鐘（西北 -12.6, -5.6）：**指針永遠停在 2:17**（可互動）
- 辦公室（西北角房 x ∈ [-12.8,-8.1]，z ∈ [-6,-2.55]，門在東牆）：證據①所在
- 入口雙木門（南牆中央，進場後鎖死）、兩側高窗、吊燈、休息椅、散落行李

## 7. 黑悟空風格光影（已寫入 DefaultEngine.ini 的基礎上，場景內手調）
- PostProcessVolume（Unbound）：曝光補償 -0.5、Filmic 色調、Bloom 強度 0.6、Vignette 0.5、Film Grain 0.35、色差 0.3、暗部偏冷 (0.95,0.97,1.06)/亮部偏暖 (1.05,1.0,0.94)
- ExponentialHeightFog：密度 0.06、體積霧開、散射 0.4
- 手電筒（角色 SpotLight）：5000 lm、內錐 18°/外錐 26°、色溫 4200K、**Volumetric Scattering 2.0**（塵霧光柱）、投影開
- 壁燈：1200 lm 鎢絲色 2700K，藍圖 Timeline 閃爍（0.5–0.9 隨機抖動，偶發瞬滅 0.1s）
- 月光（窗）：冷色 Rect Light 8000K 微弱滲入
- 走廊保持「手電筒外幾乎全黑」——恐怖感來自黑暗預算，不要提亮環境光
